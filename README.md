# Groovy-Example

Examples of Groovy and Gradle codes.

## Features

* Develop with Eclipse
* Multiproject structure
* Tests with Spock
* Code coverage with Cobertura

## Setup

* JDK 7
* Eclipse 4.4 Luna
* Groovy-Eclipse (for 4.4)
* Gradle IDE Pack
* Set `GRADLE_USER_HOME` classpath variable to `/path/to/home/.gradle`
